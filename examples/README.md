# 示例数据目录

本目录包含平台的示例数据，用于演示和测试。

## 数据文件

### demands.json - 需求数据
包含示例需求数据，展示平台支持的各类需求场景：
- 志愿服务
- 教育公益
- 创业招募
- 校园公益
- 社区服务

### resources.json - 资源数据
包含示例资源数据，展示平台支持的各类资源类型：
- 场地资源
- 设备资源
- 服务资源
- 物资资源
- 教育资源

### users.json - 用户数据
包含示例用户数据，展示平台支持的用户类型：
- 公益组织
- 创业团队
- 社区团体

## 使用方式

这些数据可用于：
1. 前端开发时的模拟数据
2. API 接口测试
3. 功能演示
4. 单元测试

## 数据结构说明

详细的数据结构说明请参阅 [API 文档](../docs/api/README.md)。
